# Online Quiz Application in C

## Project Summary
This project is a simple console-based Online Quiz Application developed in C.  
It allows users to add multiple-choice questions, view all stored questions, conduct a quiz, and view the final score.  
The project is structured using separate source files and header files as per UPES major project guidelines.

## Concepts Demonstrated
- Structures
- Functions and modular programming
- Arrays and string handling
- Menu-driven programming
- Header files and separate compilation

## How to Run
./quiz
